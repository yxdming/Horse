## 🐛 常见问题排查

### 前端无法访问
```bash
# 检查端口占用
netstat -ano | findstr :5174

# 重启前端服务
cd d:\aidp\frontend-vue
npm run dev
```

### 后端API调用失败
```bash
# 检查后端服务
netstat -ano | findstr :8000

# 测试API
curl http://localhost:8000/health

# 查看后端日志
cd d:\aidp\backend
# 查看运行日志
```

### 登录失败
- 确认后端服务运行正常
- 检查用户数据是否存在
- 查看浏览器控制台错误信息

### 页面白屏
- 打开浏览器开发者工具（F12）
- 查看Console标签的错误信息
- 检查Network标签的API请求

### 图表不显示
- 检查ECharts是否正确安装
- 确认图表数据格式正确
- 查看Console是否有错误

## 📊 性能指标

### 首屏加载
- 目标: < 2秒
- 测试方法: 清除缓存后刷新页面，记录加载时间

### 路由切换
- 目标: < 500ms
- 测试方法: 在不同页面间切换，观察响应速度

### API响应
- 目标: < 1秒
- 测试方法: 在Network标签查看API请求时长
