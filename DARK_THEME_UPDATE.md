
## 💡 使用建议

### 1. 亮度和对比度
- 确保文字清晰可读
- 使用半透明白色而非纯白
- 保持足够的对比度

### 2. 品牌色
- 主色调: `#177ddc` (蓝色)
- 成功色: `#52c41a` (绿色)
- 警告色: `#faad14` (橙色)
- 错误色: `#ff4d4f` (红色)

### 3. 自定义主题
如需调整，修改 `App.tsx`:
```typescript
theme={{
  algorithm: [theme.darkAlgorithm],
  token: {
    colorPrimary: '#你的颜色',
    colorBgBase: '#背景色',
    // ...
  },
}}
```
